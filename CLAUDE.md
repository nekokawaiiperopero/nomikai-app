# 飲み会アプリ - プロジェクト設定

## 概要
飲み会の幹事（ホスト）がAIエージェントと対話して飲み会を企画し、
参加者への連絡・やりとりをAIに委任するWebアプリケーション。

## 主要画面
- S1: トップ画面（PJ作成入口）
- S2: ホストチャット画面（AIと対話して要件確定・参加者登録）
- S3: ダッシュボード画面（参加者一覧・状況管理）
- S4: 参加者チャット画面（AIと対話して参加可否回答）

## ドキュメント
- docs/05_ユーザー要求ドキュメント.md - 課題・ユーザーストーリー・利用フロー
- docs/06_要件定義書.md - 機能要件・データ設計・画面遷移

## 技術スタック
- HTML / CSS / JavaScript（単一HTMLファイル構成）
- 外部ライブラリはCDN経由のみ
- データ永続化は localStorage

## 開発ルール
- UIは日本語
- レスポンシブ対応必須（モバイルファースト）
- CSS変数でカラー管理、ダークモード対応推奨
- 関数には日本語コメント必須
- localStorageキーは `nomikai-app-` プレフィックスを使用

## フォルダ構成
```
飲み会アプリ/
├── CLAUDE.md          # このファイル（プロジェクト設定）
├── .claude/           # Claude Code設定
│   └── launch.json    # プレビューサーバー設定
├── app/               # アプリ本体（HTMLファイル群）
├── docs/              # プロジェクト解説文書
└── .gitignore         # Git除外設定
```

## コマンド
- プレビュー: `.claude/launch.json` の `nomikai-app` で起動
- HTMLファイルはブラウザで直接開いても動作する
