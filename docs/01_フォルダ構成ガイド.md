# フォルダ構成ガイド

## 全体構成

```
飲み会アプリ/
├── CLAUDE.md              ... (1) プロジェクト設定ファイル
├── .claude/               ... (2) Claude Code 設定ディレクトリ
│   └── launch.json        ... (2a) プレビューサーバー定義
├── app/                   ... (3) アプリ本体
│   └── (HTMLファイル群)
├── docs/                  ... (4) 解説文書ディレクトリ
│   ├── 01_フォルダ構成ガイド.md
│   ├── 02_CLAUDE_md解説.md
│   ├── 03_Claude_Code機能活用ガイド.md
│   └── 04_開発ワークフロー.md
├── .gitignore             ... (5) Git除外設定
└── .git/                  ... (6) Gitリポジトリ
```

---

## 各要素の解説

### (1) CLAUDE.md - プロジェクト設定ファイル
Claude Codeが**最初に読み込む設定ファイル**。プロジェクトのルールや技術スタック、
コーディング規約を定義する。Claude Codeはこのファイルの内容を毎回の会話で参照し、
プロジェクトに一貫した開発を行う。

### (2) .claude/ - Claude Code 設定ディレクトリ
Claude Code固有の設定を格納するディレクトリ。

#### (2a) launch.json - プレビューサーバー定義
`preview_start` 機能で使うローカルサーバーの設定。
Pythonの `http.server` を使い、`app/` ディレクトリをポート3001で配信する。
これにより、Claude Codeが自動でブラウザプレビューを起動し、
画面を確認しながら開発できる。

### (3) app/ - アプリ本体
HTMLファイルを格納するディレクトリ。
開発スタンダードに従い、各機能は**単一HTMLファイル**にまとめる（HTML+CSS+JS一体）。
例: `app/index.html`, `app/vote.html`, `app/timer.html`

### (4) docs/ - 解説文書
プロジェクトの解説やガイドを格納するディレクトリ。
開発者がプロジェクトの全体像を理解するためのドキュメント群。

### (5) .gitignore - Git除外設定
バージョン管理から除外するファイルのパターンを定義。

### (6) .git/ - Gitリポジトリ
バージョン管理用。変更履歴の追跡やブランチ管理に使用。
